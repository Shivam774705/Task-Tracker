# Task Tracker – Full Stack Assignment

A simple and responsive **Task Tracker Web Application** developed using **HTML, CSS, and JavaScript** as part of the **Full-Stack Technical Assignment** for **Nova Technology**.

This project demonstrates core frontend development skills, clean UI design, and proper state handling without using any external frameworks.

---

## Assignment Objective

The objective of this assignment was to build a simple web application called **“Task Tracker”** that allows users to:

- Add a task  
- View all tasks  
- Mark tasks as completed  

The implementation strictly follows the problem statement provided by the company.

---

## Features

- Add new tasks  
- View all added tasks  
- Mark tasks as **Pending** or **Completed**  
- Delete tasks  
- Filter tasks by date  
- Filter tasks by status (All / Pending / Completed)  
- Session-based data persistence  
- Fully responsive design (Mobile, Tablet, Desktop)

---

## Tech Stack Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and responsive layout  
- **JavaScript (Vanilla)** – Application logic and interactivity  
- **Session Storage** – Temporary task persistence  
- **Remix Icons** – User interface icons  

> No external frameworks or libraries were used, as per assignment requirements.
